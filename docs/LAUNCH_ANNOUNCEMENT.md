## 📢 Press Release

### BTC Collateral Monitor Launches as Free Open Source Tool for Rootstock DeFi Users

**[City], [Date]** — Today marks the launch of BTC Collateral Monitor, a free and open-source service designed to help Rootstock (RSK) DeFi users protect their BTC-backed positions from liquidation.

The tool monitors collateral ratios in real-time and sends instant Telegram alerts when positions approach dangerous levels, giving users time to add collateral or close positions before liquidation occurs.

### Key Features

- **Real-Time Monitoring:** Checks positions every 10 minutes
- **Instant Alerts:** Telegram notifications for warning, critical, and liquidation levels
- **Privacy-Focused:** Only users can see their own data
- **100% Free:** No subscriptions, no premium tiers, completely open source
- **Self-Hostable:** Users can run their own instance

### Built for the Community

"I built this because I saw too many people getting liquidated during market volatility," said Matias Fleker, creator of BTC Collateral Monitor. "Everyone should have access to tools that help them manage risk, regardless of their budget."

The project is released under the MIT License, allowing anyone to use, modify, and contribute to the codebase.

### Technical Details

BTC Collateral Monitor is built with:
- Python and Web3.py for blockchain interaction
- Telegram Bot API for user communication
- Streamlit for the dashboard
- PostgreSQL for data storage
- Docker for easy deployment

Currently supports Money on Chain protocol, with plans to add more RSK DeFi protocols in the future.

### Availability

The service is available now:
- **Telegram Bot:** @RSKCollateralBot
- **Dashboard:** https://rskcollateral.com
- **Source Code:** https://github.com/oVservant/rsk-collateral-monitor

### About Rootstock

Rootstock (RSK) is a Bitcoin sidechain that enables smart contracts and DeFi applications while maintaining Bitcoin's security. It has become a hub for BTC-backed DeFi, with protocols like Money on Chain allowing users to borrow stablecoins against BTC collateral.

### About BTC Collateral Monitor

BTC Collateral Monitor is a community-driven open source project focused on improving risk management tools for DeFi users. The project is funded by donations and run by volunteers.
